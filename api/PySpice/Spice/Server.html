3.1.7.8. Server¶

class PySpice.Spice.Server.SpiceServer(spice_command='ngspice')[source]¶

Bases: object

This class wraps the execution of Spice in server mode and convert the output in Python data structure.

Example of usage:

spice_server = SpiceServer(spice_command='/path/to/ngspice')
raw_file = spice_server(spice_input)

It returns a PySpice.Spice.RawFile instance.
